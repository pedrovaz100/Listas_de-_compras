## 🌐 Endpoints

### Listas

- GET /listas  
- GET /listas/{id}  
- POST /listas  
- PUT /listas/{id}  
- DELETE /listas/{id}  

### Itens

- GET /itens  
- GET /itens/{id}  
- POST /itens  
- PUT /itens/{id}  
- DELETE /itens/{id}  

### Outros

- GET /health
