# 🛒 Lista de Compras API

API desenvolvida com Spring Boot para gerenciar listas de compras e itens.

---

## 🚀 Tecnologias

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Lombok

---

## ▶️ Como rodar

```bash
mvn spring-boot:run
```

---

## 🌐 Endpoints

- GET /listas  
- POST /listas  
- PUT /listas/{id}  
- DELETE /listas/{id}  

- GET /itens  
- POST /itens  

---

## 🧪 H2 Console

Acesse:

```
http://localhost:8080/h2-console
```
